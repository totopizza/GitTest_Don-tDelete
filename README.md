# gitの使い方

## gitのクローン
* git clone Git URL
 - Git上にあるリポジトリをローカル上にコピーする。

## ファイルの追加
* git add File or Folder
 - 指定したファイルフォルダを追加する。

## コメント
* git commit -m" ～ "
 - 追加したファイル・フォルダに対してコメントする。

## アップする
* git push origin master
 - ローカルgitをネット上のGitに対してアップする。

### 補足説明
* ブランチ
 - git branch ブランチを作る
 - git checkout ブランチを切り替える
